Student Management
- A web application that allows teachers to easily manage their students.




Features

- Secure signup and login (using bcrypt)
- Edit profile and password
- Add, delete, and update students
- View the full list of students




Technologies Used

- Frontend: React.js, CSS
- Backend: Node.js, Express.js
- Database: MongoDB (Mongoose)






Gestion des Étudiants
Application web permettant aux professeurs de gérer leurs étudiants facilement.



Fonctionnalités
- Inscription et connexion sécurisées (bcrypt) 
- Modification du profil et du mot de passe 
- Ajout, suppression et modification des étudiants 
- Visualisation de la liste des étudiants



Technologies utilisées
- Frontend : React.js, CSS 
- Backend : Node.js, Express.js
